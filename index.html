게시판 프로젝트


html과 JSP를 이용한 프로젝트입니다. 누구나 자유롭게 게시글을 작성할 수 있는 페이지를 만들고자 하였습니다.

외부 자료에서 참고한 것이 많지만, Bootstrap 등을 활용하여 일반적인 html 페이지처럼 보이는 것이 아니라 좀 더 미관상 깔끔하면서도 제작한 사람의 특징이 묻어나오는 페이지를 제작하도록 노력하였습니다.

글 작성, 수정, 삭제, 답변 게시 등의 기능을 구현하였으며 수정, 삭제 시에는 게시글을 작성할 때에 입력한 비밀번호와 동일한 비밀번호를 입력해야만 해당 작업을 수행할 수 있도록 하였습니다.

게임 매장 시뮬레이터


C#의 윈폼을 활용하여 제작하였으며, 각종 비디오 게임을 판매하는 매장에서 사용하는 프로그램이라는 컨셉을 잡아 제작한 프로그램입니다.

실존하는 게임의 타이틀을 입력하여 데이터를 추가하였고, 별도의 파일을 통해 데이터를 통제하기에 DB연동을 따로 지원하지 않습니다. 프로그램 내에서 상품을 추가할 수 있으며 상품 판매 또한 가능합니다. 상품을 판매할 시에는 물품의 가격에 따라 수익이 늘어나며, 물품의 가짓수는 구매한만큼 줄어들 수 있도록 구현하였습니다.
